Basic Steps to assembling a keyboard

Assembly Steps

  1. Lube stabalizers
  2. *Lube switches
  3. Attach stabalizers to pcb
  4. Insert switches into plate
  5. Join plate with pcb
  6. Solder switches
  7. *Mount/solder LEDs
  8. Join pcb/plate to case
  9. Insert keycaps appropriately
  10. Program keyboard with desired layout

Necessary Components

  • Lubrication
  • Plate
  • Switches
  • Stabalizers
  • PCB
  • Case
  • Hardware
  • Keycaps
  • USB cable
  • *LEDs
